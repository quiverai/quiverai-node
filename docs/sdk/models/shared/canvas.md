# Canvas

Controls the coordinate system and "Safe Zones" to prevent clipping/bad cropping.
NOTE: "mode" determines the default canvas if this is omitted.


## Example Usage

```typescript
import { Canvas } from "@quiverai/sdk/sdk/models/shared";

let value: Canvas = {
  safeZone: {},
  viewBox: {},
};
```
