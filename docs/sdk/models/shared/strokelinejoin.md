# StrokeLineJoin

The shape at the corners where paths meet.
- `miter`: Sharp pointed corner
- `round`: Rounded corner
- `bevel`: Flat diagonal corner


## Example Usage

```typescript
import { StrokeLineJoin } from "@quiverai/sdk/sdk/models/shared";

let value: StrokeLineJoin = "round";
```
