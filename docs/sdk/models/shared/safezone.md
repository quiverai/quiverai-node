# SafeZone

The "Safe Zone" is the guaranteed empty margin between content and edge.
Prevents clipping and bad cropping.


## Example Usage

```typescript
import { SafeZone } from "@quiverai/sdk/sdk/models/shared";

let value: SafeZone = {};
```
